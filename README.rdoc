== WORDSHOP

Inspired by genius.com, Wordshop is a tool for writers to give and receive feedback on their work. 
The use of inline annotations allows users to leave critiques on specific portions of text. 
The live site can be accessed at http://wordshop.xyz

Wordshop is a single-page Backbone.js application consuming a RESTful Rails JSON API. 

Tools/Libraries used:
* typeahead.js
* bootbox.js
* moment.js
* serializeJSON
* filepicker.io
